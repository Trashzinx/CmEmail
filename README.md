<h2> Descrição: </h2>

<p>Script feito para Automatizar o envio de emails utilizando o mutt</p>

<h2> Instalação: </h2>

```bash
apt-get update -y && apt-get upgrade -y

apt-get install git

git clone https://github.com/AthenaD3V/CmEmail

cd CmEmail

chmod +x *
```

<h3> Inicialização: </h3>

```bash
./install

# depois

./CmEmail
```
